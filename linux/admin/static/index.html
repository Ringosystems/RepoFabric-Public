    Flips sign-in to Entra-only and clears the sandbox warnings. The local admin becomes a dormant break-glass.
    Edits below are split across service.yaml (operational knobs) and solution.yaml (auth, targets, notifications). A timestamped backup of each file is written before save. In-flight operations keep the old config; new bridge calls pick up changes after the next module reload.
    
  
    Backup & disaster recovery
    
  
    
    
    SQLite holds a content-addressed archive of every commit on every virtual repo. Daily snapshots run at 02:00 UTC; publishes, promotes, reverts, and drift captures also write snapshots inline. Verify-backup reconstructs a snapshot into a temporary bare repo, runs git fsck, and checks the head SHA matches byte-for-byte.

    Retention cleanup
    
  
    
    
    Each subscription keeps its latest Retention versions (plus any pinned); a daily sweep at 02:30 (container time) prunes older versions from the repo and deletes their installers. Run it now to bring the Catalog's Pubs counts down to each row's Retention without waiting for the nightly job.
    
    
    
    
    
      
    
    
  
    
  
    Search popularity
    
  
    
    
    Search results are ranked by popularity data fetched from winget.run on a daily cron. This is a request-count proxy (not real install telemetry) and is good enough for "well-known apps appear first". Set popularity.disabled: true in service.yaml to turn off and revert to alphabetical sort.
