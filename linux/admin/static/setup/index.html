Welcome

The container is in first-run setup. Find the one-time setup token in one of two places:

  • docker logs repofabric-linux — printed at boot when no config exists yet
  • /var/lib/repofabric/setup-token.txt on the host

Identity (Microsoft Entra)

Admins sign in with Microsoft Entra. RepoFabric needs an app registration in your tenant; its redirect URI must be <your-public-url>/admin/auth/callback. Don't have one yet? Generate the commands below.

No app registration yet? Generate the setup commands

Creating the app registration requires a Global Administrator or Privileged Role Administrator — that's an Entra rule no tool can skip. The no-install way is Azure Cloud Shell (you're already signed in there):

  1. Click Generate commands, then copy the script.
  2. Open shell.azure.com (sign in as an admin), paste, and run it.
  3. Paste what it prints into the box below to autofill, or type the three values into the fields yourself.

        Admin access full control
        
Who can change RepoFabric: add subscriptions, publish, sync, edit settings. Point this at an Entra group so access follows your existing joiner/mover/leaver process; list individual UPNs only for exceptions.

        
        
      

      

        Read-only access view only
        
Who can sign in and see everything but change nothing (helpdesk, auditors, managers). Every write is refused for these accounts. Again, prefer a group.
